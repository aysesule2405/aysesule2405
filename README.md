# Featured Projects
 
## 👁️ Aperture · Slack Accessibility Agent
### *Slack Agent Builder Challenge 2026 · Agent for Good*
 
An AI agent that gives every image in Slack a voice. Almost no images posted to Slack carry alt text, so a screen reader announces only "image dot png" and the content inside is lost. Aperture closes that gap automatically, without asking anyone to change how they work.
 
### Features
- Automatic screen-reader-ready descriptions posted in-thread the moment an image is shared
- Chart-to-data extraction that pulls real values out of a chart into a readable table, so a blind analyst gets the numbers, not just "a bar chart"
- Context grounding through Slack's Real-Time Search API, so a description explains why an image matters in its conversation
- Six-verb command set: `describe`, `catch me up`, `simpler`, `translate`, `backfill`, `help`
- History backfill that sweeps a channel so nothing stays invisible
- SQLite caching so no image is ever described twice
- Runs entirely on free tiers with no hosting required
### Accessibility Engineering
Descriptions open with a single labeling word, stay brief by default to reduce listening fatigue, never encode meaning in color alone, and fail with one honest line instead of going silent. The screen-reader delivery path was validated with VoiceOver before any feature work began.
 
### Stack
`Bolt for Python` `Slack Real-Time Search API` `Block Kit` `Socket Mode` `Gemini Flash` `SQLite`
 
🔗 [GitHub](https://github.com/aysesule2405/aperture)
🔗 [Demo Video](https://www.youtube.com/watch?v=j3dVx-kKC4A)
