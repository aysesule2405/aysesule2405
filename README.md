### 👁️ Aperture — Slack Accessibility Agent
*Slack Agent Builder Challenge 2026*
 
An AI agent that gives every image in Slack a voice. Most images posted to Slack carry no alt text, so a screen reader announces only "image dot png." Aperture closes that gap automatically.
 
- Automatic screen-reader-ready descriptions posted in-thread the moment an image is shared
- Chart-to-data extraction, pulling real values out of a chart into a readable table
- Context grounding via Slack's Real-Time Search API
- Six-command set: `describe` `catch me up` `simpler` `translate` `backfill` `help`
- Validated with VoiceOver before feature work began; brief by default, never color-only, fails with one honest line instead of going silent

### 🎥 Framewise — AI Video Workspace + Chrome Extension
*HackHCC 2026 · Best Use of AI & Best Use of MongoDB Atlas*
 
Turns any browser video into a searchable workspace: a React web app and Chrome extension sharing one backend, so notes and understanding carry over from whatever tab you're watching in.
 
- React 18 app + Chrome MV3 extension sharing one Node.js/Express backend and MongoDB Atlas database
- Gemini 2.5-powered Q&A, timestamped notes, and quizzes generated directly from video content
- On-device BiLSTM pose classifier (Keras → int8 → TensorFlow.js, ~2MB) for real-time skeletal overlay with sub-frame latency and zero network dependency
- Instrumented with Prometheus + Grafana for production observability
